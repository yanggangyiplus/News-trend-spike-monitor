# Assets 디렉토리

이 디렉토리는 프로젝트 문서에서 사용하는 이미지 및 다이어그램을 저장합니다.

## 파일 설명

### 아키텍처 다이어그램
- `architecture.png`: 전체 시스템 아키텍처 다이어그램
- `pipeline_overview.png`: 데이터 파이프라인 개요도

### 대시보드 데모
- `dashboard_demo.gif`: Streamlit 대시보드 데모 영상
- `dashboard_screenshot.png`: 대시보드 스크린샷

### 성능 메트릭
- `performance_metrics.png`: 성능 메트릭 시각화
- `scalability_diagram.png`: 확장성 다이어그램

## 사용 방법

README.md 및 문서에서 다음과 같이 참조:

```markdown
![시스템 아키텍처](assets/architecture.png)
```

## 이미지 생성 가이드

### 아키텍처 다이어그램
- 도구: draw.io, Lucidchart, Mermaid
- 형식: PNG 또는 SVG
- 해상도: 최소 1920x1080

### 대시보드 스크린샷
- 형식: PNG
- 해상도: 최소 1920x1080
- 파일 크기: 최대 2MB

### 데모 영상
- 형식: GIF 또는 MP4
- 길이: 최대 30초
- 파일 크기: 최대 5MB
